# Content Classifier Data

## Dataset composition

The initial scaffold uses a small, curated sample based on the proposed architecture:

- 50–60% real/public + curated examples
- 25–35% synthetic examples
- 10–15% hard/boundary examples

This folder is intended to grow into a larger, provenance-aware dataset over time.

## Schema

Columns:

- id
- text
- label
- source
- source_url
- collected_at
